# Weather Diary (Дневник погоды)

**Автор:** Красиков Дмитрий

## Описание программы
**Weather Diary** — это настольное приложение с графическим интерфейсом (GUI), предназначенное для ведения персонального журнала погодных наблюдений. Программа позволяет пользователю фиксировать ежедневные данные о температуре, типе погоды и наличии осадков. Все данные надежно сохраняются в формате JSON, что обеспечивает их сохранность между сессиями работы.

**Основные возможности:**
*   Добавление записей с проверкой корректности (валидацией) ввода.
*   Просмотр истории записей в удобной таблице.
*   Фильтрация данных по температуре (поиск записей выше заданного значения).
*   Автоматическое сохранение и загрузка базы данных из файла `weather_data.json`.

## Инструкция по запуску

### 1. Установка зависимостей
Программа написана на языке Python с использованием встроенной библиотеки **Tkinter**. 
*   Если у вас установлен Python, устанавливать дополнительные библиотеки не требуется.
*   Если библиотека Tkinter отсутствует (актуально для некоторых дистрибутивов Linux), установите её командой:
    ```bash
    sudo apt-get install python3-tk
    ```

### 2. Запуск программы
1.  Скачайте файл с кодом `weather_diary.py` в рабочую директорию.
2.  Откройте терминал или командную строку в этой папке.
3.  Введите следующую команду для запуска:

```bash
python weather_diary.py
```

## Примеры использования и работы

### Добавление новой записи
1.  В поле **"Дата"** введите: `20.05.2024`.
2.  В поле **"Температура"** введите число: `22`.
3.  В поле **"Описание"** введите: `Ясно, легкий ветер`.
4.  Отметьте галочку **"Осадки"**, если они были.
5.  Нажмите кнопку **"Добавить запись"**. Запись мгновенно появится в таблице снизу.

### Фильтрация данных
*   Чтобы найти все теплые дни, введите в поле фильтра «Мин. Температура» число `10` и нажмите **"Применить фильтр"**. В таблице останутся только те дни, где температура была +10°C и выше.
*   Для возврата к полному списку нажмите кнопку **"Сброс"**.

### Работа с файлами
*   При закрытии программы все данные автоматически сохраняются в файл `weather_data.json` в папке с программой. При следующем запуске программа сама считает эти данные и выведет их в таблицу.
